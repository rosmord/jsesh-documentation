Le immagini sui computer si dividono in due categorie: *bitmap* e immagini *vettoriali*. Per avere un'idea di cosa siano, diciamo che le bitmap vengono solitamente modificate con software come Photoshop (o GIMP se si usa software libero), mentre le immagini vettoriali vengono modificate con software come Adobe Illustrator (o Inkscape).

La maggior parte dei software può visualizzare e leggere le bitmap, quindi tendono ad essere più facili da condividere. Il loro problema principale è che non è possibile ingrandirle correttamente. Le bitmap sono fondamentalmente mosaici composti da piccoli punti quadrati, chiamati *pixel*; se si ingrandisce, quei piccoli punti quadrati diventano grandi quadrati. Una bitmap adatta alla visualizzazione a schermo probabilmente verrà stampata male; una bitmap adatta alla stampa dovrà avere una risoluzione molto alta. Tuttavia, condividere immagini vettoriali a volte è difficile (senza una buona ragione, ma è una situazione spiacevole), quindi si potrebbe finire per inviare bitmap ad alta risoluzione per la stampa. **Nota del 2025: le cose vanno molto meglio ora**.

Per esportare una bitmap, seleziona la voce di menu "File/Esporta come/Esporta come bitmap".

Puoi esportare una versione jpeg o png del testo. Se non è selezionato alcun testo, verrà visualizzato l'intero testo. Se è presente una selezione, verrà visualizzato solo il testo selezionato.
