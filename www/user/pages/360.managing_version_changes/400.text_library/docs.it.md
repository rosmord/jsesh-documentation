JSesh include una libreria di testo in una cartella chiamata `texts`. È inclusa nella distribuzione del software JSesh, il che significa che la cartella relativa alla versione *n* verrà eliminata quando si disinstalla questa versione.

Consiglio **fortemente**, se si desidera lavorare su un testo, di **copiare** il file da qualche altra parte, ad esempio nella cartella dei documenti.

Si noti che i testi hanno ora una propria cartella su github: [MDC-texts](https://github.com/rosmord/MDC-texts).
