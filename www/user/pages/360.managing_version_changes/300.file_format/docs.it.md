La compatibilità dei file mi sta molto a cuore. Uno dei primi vantaggi di JSesh, negli anni 2000, era la possibilità di leggere con buoni risultati sia i file Winglyph che Macscribe (oltre al mio dialetto *Manuel de Codage*, per tksesh).

In teoria, un file **scritto** dalla versione *n* di JSesh dovrebbe essere leggibile da tutte le versioni successive di JSesh. Il contrario non è vero: se aggiungo una nuova funzionalità, non sarà supportata dalle versioni precedenti!

Se a un certo punto doveste riscontrare problemi di compatibilità con i file JSesh, **contattatemi** (ovviamente, fate regolarmente backup dei vostri file, non solo della versione corrente, giusto?).
