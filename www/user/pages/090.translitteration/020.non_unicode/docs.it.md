**TI CONSIGLIO VIVAMENTE DI USARE UNICODE** (Scusate se mi sono lamentato, ma è importante).

Puoi selezionare un font con una codifica compatibile con MdC. In questo caso, JSesh non gestisce le lettere maiuscole. Puoi scegliere un font personalizzato, se ne hai uno, oppure utilizzare il font di traslitterazione predefinito di JSesh. Una versione di questo font è integrata nel software, ma puoi installarla sul tuo sistema (ad esempio per utilizzarla con un elaboratore di testi). Una copia di questo font è disponibile nella cartella dei font dell'installazione di JSesh.

## Metodo semplice con il vecchio font

Dovresti aprire il menu "Preferenze" (su Mac, si trova nel menu "JSesh"; altrove, si trova nel menu "Modifica"). Vai alla scheda "Preferenze font" e clicca sul pulsante "Usa il vecchio font di traslitterazione".

![](./fontprefsmdctranslit.png)

Ecco fatto. Per i tuoi documenti Word (o OpenOffice...), puoi trovare il font nella cartella dell'applicazione JSesh (nella cartella "font").

## Utilizzo dei tuoi font compatibili con MdC

Se disponi di un font compatibile con MdC (ovvero, la "a" rappresenta una ayn, la "A" rappresenta una aleph, ecc.), puoi utilizzarlo.

Seleziona semplicemente il font (utilizzando "Scegli font" davanti al font di traslitterazione) e poi, nelle opzioni avanzate, seleziona "Font di traslitterazione MdC (non Unicode)".
